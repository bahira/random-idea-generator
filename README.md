# AIdea Generator

An AI-powered tool to spark creativity and generate innovative ideas instantly. Perfect for creators, entrepreneurs, and anyone looking to break through creative blocks.

## Features

- **Instant Idea Generation**: Get creative ideas in seconds
- **Categorized Ideas**: Choose from Creative, Business, or Personal Growth categories
- **Save Ideas**: Keep track of your favorite ideas
- **Modern UI**: Beautiful, responsive design built with Next.js and Tailwind CSS
- **Free Tier**: Generate up to 5 ideas for free, upgrade for unlimited access

## Getting Started

First, install dependencies:

```bash
npm install
# or
yarn install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Geist Font
